FAQ

Questions fréquentes

Mes données sont-elles envoyées dans le cloud ? →
Non. AuditM365 est une application Windows locale. Toutes les données restent sur votre machine. La connexion à Microsoft 365 se fait directement via Microsoft Graph API.
La licence est-elle liée à un seul PC ? →
Oui, chaque licence est liée à une machine. Si vous changez de PC, contactez-nous pour un reset gratuit de votre licence.
Puis-je tester avant d'acheter ? →
Oui, nous proposons une licence de test d'1 jour. Contactez-nous à contact-licence@m365intune.com pour en obtenir une.
Quels droits Microsoft 365 sont nécessaires ? →
Un compte avec les droits de lecture sur votre tenant suffit. AuditM365 utilise Microsoft Graph avec des permissions en lecture seule uniquement.
Les mises à jour sont-elles incluses ? →
Oui, toutes les mises à jour sont incluses dans votre licence. Nouvelles fonctionnalités, correctifs et améliorations sont disponibles sans frais supplémentaires.
Quel format pour les rapports exportés ? →
AuditM365 génère des rapports en HTML (lisibles dans n'importe quel navigateur) et en Excel, prêts à partager avec vos clients ou votre direction.
