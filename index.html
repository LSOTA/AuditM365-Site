FAQ

Questions fréquentes

🔒 Mes données M365 sont-elles envoyées dans le cloud ?
Non. AuditM365 est une application Windows locale. Toutes les données restent sur votre machine. La connexion à Microsoft 365 se fait directement via Microsoft Graph API en lecture seule.
💻 La licence est-elle liée à un seul PC ?
Oui, chaque licence est liée à une machine pour des raisons de sécurité. Si vous changez de PC, contactez-nous pour un reset gratuit de votre licence sous 24h.
🧪 Puis-je tester avant d'acheter ?
Oui, nous proposons une licence de test d'1 jour. Contactez-nous à contact-licence@m365intune.com pour en obtenir une gratuitement.
🔑 Quels droits M365 sont nécessaires ?
Un compte admin M365 suffit pour la connexion initiale. AuditM365 crée automatiquement l'App Registration avec des permissions en lecture seule uniquement.
🔄 Les mises à jour sont-elles incluses ?
Oui, toutes les mises à jour sont incluses dans votre licence. Nouvelles fonctionnalités et correctifs disponibles sans frais supplémentaires.
📊 Quels formats pour les rapports exportés ?
AuditM365 génère des rapports en HTML (lisibles dans tout navigateur) et en Excel, prêts à partager avec vos clients ou votre direction.
